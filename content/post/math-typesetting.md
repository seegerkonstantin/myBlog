Mathematical notation in a Hugo project can be enabled by using third party JavaScript libraries.

### Examples

```

This is inline $ \(\varphi = \dfrac{1+\sqrt5}{2}= 1.6180339887…\)$ math.

This is block math:

$$
 \varphi = 1+\frac{1} {1+\frac{1} {1+\frac{1} {1+\cdots} } } 
$$
```
